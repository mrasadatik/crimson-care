\begin{Indent}\textbf{ load\+Hospitals}\par
{\em Load hospitals from file

This function loads the hospitals from the file {\ttfamily resources/db/hospitals.\+txt} and stores it in the {\ttfamily hospital\+Head} linkedlist.

\begin{DoxyNote}{Note}
The file path \textquotesingle{}resources/db\textquotesingle{} is relative to the project root directory. Make sure that the folder exists also to run the program from the root directory.
\end{DoxyNote}
\begin{DoxyPostcond}{Postcondition}
If the file is not found, the function does nothing. Otherwise, the hospitals are loaded from the file and stored in the {\ttfamily hospital\+Head} linkedlist.
\end{DoxyPostcond}
