Example\+:

{\bfseries{On Linux/\+Mac}}


\begin{DoxyItemize}
\item Debug Build\+:

{\ttfamily bash ./bin/\+Debug/\+Crimson\+Care }
\item Release Build\+:

{\ttfamily bash ./bin/\+Release/\+Crimson\+Care }
\end{DoxyItemize}

{\bfseries{On Windows}}


\begin{DoxyItemize}
