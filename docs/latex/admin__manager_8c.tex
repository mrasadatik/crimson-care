\begin{Indent}\textbf{ load\+Admin\+Credentials}\par
{\em Load admin credentials from file

This function loads the admin credentials from the file {\ttfamily resources/db/admin\+\_\+credentials.\+dat} and stores it in the {\ttfamily admin\+Head} linkedlist. If file is not found, it creates a new admin with default credentials and stores it in the file.

\begin{DoxyNote}{Note}
The file path \textquotesingle{}resources/db\textquotesingle{} is relative to the project root directory. Make sure that the folder exists also to run the program from the root directory.
\end{DoxyNote}
\begin{DoxyPostcond}{Postcondition}
If the file is not found, a new admin is created with default credentials and stored in the file. If the file is found, the admin credentials are loaded from the file and stored in the {\ttfamily admin\+Head} linkedlist.
